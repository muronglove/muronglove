sublime快捷键

cmd类

  • cmd+l— delete the line
  • cmd+delete—删除整行
  • cmd+shift+d—复制光标所在行并粘贴到下一行
  • cmd+/—注释一行，或者注释所选内容
  • cmd+alt+/—取消注释
  • cmd+[]—整体前进或后退
  • cmd+kk—删除光标后的一行代码
  • cmd+z—撤销
  • cmd+y—恢复撤销
  • cmd+d—更改变量名
  • cmd+d,cmd+k—跳过某个变量名
  • cmd+shift+v—自适应缩进的粘贴
  • cmd+shift+k—选中一组标签

其他类

  • shift+[]—括号补全或删除
  • 键入fz3.6/2：/按Cmd-Shift-Y (Win: Ctrl-Shift-Y)计算数值：/然后按Tab展开Snippet：

sublime插件快捷键

一些插件的快捷键

安装的插件有：

  • JsFormat,全选之后，右键jsFormat即可对js进行排版
  • package control，使用shift+cmd+p，调出package control，选择install，然后查找插件进行安装
  • Alignment,cmd+l连续选中几行，然后control+option+a代码对齐
  • 快捷键和主题可以通过preference->settings-user进行设置，是json格式
  • colorPicker,cmd+shift+c进行取色
  • SideBarEnhancements,这个插件可以给SublimeText的边栏菜单带来扩充的功能，包括：在当前工程文件夹中新建文件，移动文件或文件夹，产生文件或文件夹的副本，在新窗口或浏览器中打开，刷新等。
  • cssComb,让css更加美观，选中css，然后右键run csscomb即可
  • terminal,Open Terminal at File Press ctrl+shift+t on Windows and Linux, or cmd+shift+t on OS X;Open Terminal at Project Folder Press ctrl+alt+shift+t on Windows and Linux, or cmd+alt+shift+t on OS X
  • canIUse,对css代码进行浏览器兼容检测，选中后右键即可
  • trimmer,去掉多余的空格

vim学习笔记

  1. esc+i equals insert
  2. Basic movement hljk h=>left,l=>right,j=>down,k=>up
  3. word movement web could combined with number w moves to the start of next word; e moves to the end of the word; and b moves to beginning of the word.
  4. number combined with web
  5. 3i+input+esc+esc equals type input 3 times.
  6. f find
  7. % check if the brackets are right
  8. $&0 the end and the beginning of line
  9. gg to the begining of the line shift+gjump to the last of the article
  10. / use /to find, and use n and N to find the next and the previous
  11. o insert in a new line.
  12. x use x to delete a letter.
  13. r use r to replace
  14. d dd is to delete all,and dw deletes the first word on the right side of your cursor
  15. . repetition with .
  16. v visual mode that can choose something
  17. :w save :q quit :q! quit without saving
  18. u use u to undo and use ctrl +r to redo
  19. :help

Reference material:

http://www.openvim.com/tutorial.html
